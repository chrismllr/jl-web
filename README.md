## how to upload images

- Make your changes
- Save
- `git add . (adds all the current changes to staging)`
- `git commit -m 'This is the message about this commit'`
- `git push`
 

##programs you'll need:

- Sublime Text 3 
- Github desktop
- -or- Terminal

- 1400px widest/tallest @ 300dpi
